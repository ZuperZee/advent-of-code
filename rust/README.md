# Advent of code 2023 in Rust

## Run

Run a specific day

```sh
cargo run -p day01
```

## Build

Build a specific day

```sh
cargo build --release -p day01
```

Build all days

```sh
cargo build --release
```

## Benchmark

Benchmark a specific day

```sh
cargo bench -p day01
```

Benchmark a specific day and part

```sh
cargo bench -p day01 "Day 01 part 1"
```

or

```sh
cargo bench "Day 01 part 1"
```

Benchmark all days

```sh
cargo bench
```
